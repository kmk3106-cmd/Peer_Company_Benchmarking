핵심 결론 6선

1예실차 −0.0% — 가정 정확도 8사 최우수
자사 보험서비스결과(1,120억)는 CSM 상각(2,058억)·RA 해소(440억) 합과 거의 완전 일치하며, 잔차(예실차)는 −5억(−0.0%)에 그침. 삼성생명(−13.4%)·한화생명(−8.2%) 등 생보 군집 대비 자사 가정·실제 정합이 압도적으로 우수함. 보험금·유지비·해지율 가정의 운영 결과 reconciliation 수준이 동업사 중 가장 신뢰 가능한 상태로 판단됨.
2사업비 마진 47.2% — 운영효율 최강
잔여계약기간 합계 기준 예상유지비(3.21조)는 예정유지비(6.80조)의 47.2% 수준. 가격책정 시 책정한 유지비의 절반만 실제 발생할 것으로 자사가 추정하고 있어, 향후 사업비 차익(positive expense variance) 누적 여지가 가장 큼. 동양 75.4% 대비 28%p, DB 47.4%·삼성화재 51.9% 대비 우월.
3서비스결과율 10.4% — 보험본업 마진 1위
보험수익(1조 804억) 대비 보험서비스결과(1,120억) 비율 10.4%는 8사 1위. 삼성생명 10.1%·동양 9.3%·한화 6.7% 대비 우월하며 손보 군집(2.8~7.9%) 대비도 압도. 자산규모는 가장 작지만 단위 보험수익당 본업 수익성이 최강임을 시사.
4위험률 마진 103.5% — 생보 군집 대비 약함
잔여 예상보험금이 잔여 위험보험료의 103.5%로 마진 절대액 −1.2조 적자. 삼성(81.9%)·한화(78.5%)·동양(89.6%) 생보 군집 대비 약하며 손보 한화손해(104.7%)와 유사. 잔여기간대 분해 결과(§5-C): 10년 이내 88.1% / 25~30년 105.5% / 30년 초과 131.3%로 장기 보장성 상품 가정이 마진 합계를 끌어올림. 사업 mix(보장성 강화 비중 높음)·30년 초과 보장 가정의 보수성 점검 필요.
5RA/BEL 1.7% — RA 산출 보수성 점검
자사 기말 RA(4,060억) ÷ BEL(부채 26.99조) ≈ 1.7%로 생보 4사 중 최저 추정. 삼성생명·한화생명·동양생명 모두 2~4% 수준으로 보고. RA는 비금융위험 보상으로, 자사 모형이 위험 자본화 강도가 군집 대비 낮음을 시사. 보험수익 중 RA 해소 비중은 4.1%(8사 1위)로 회전속도는 빠르나 절대 자본화 수준은 점검 필요.
6CSM 2.06조 — 자산규모 종속 최저
자사 기말 CSM 2.06조는 8사 최저. 단, CSM/보험계약부채 비율 7.6%는 생보 군집(6.6~9.0%) 내 정상권으로 절대규모 차이는 자산규모 종속. 당기 상각률 10.0%는 생보 평균(9.95%) 수준이며, 동업사 8.5~11.9% 군집 중간. 신계약 CSM 인식의 지속적 유입 없이 기존 잔액의 자연감소가 진행 중.
종합 판정: 자사는 운영 측 가정 정확도(예실차) · 사업비 보수성 · 본업 마진율 3개 축에서 동업사 최강. 반면 위험률 마진(보험금 가정 보수성) · RA 자본화 강도 2개 축에서는 생보 군집 대비 약점이 존재. 이는 자사의 사업 mix(보장성 강화 비중 ↑) 및 RA 산출 모형(분산 모형 등)의 구조적 특징에서 기인하는 것으로 추정되며, 30년 초과 보장 가정과 RA 신뢰수준의 재교정이 우선 권고 사항임. CSM 절대규모는 자산규모 종속변수로, 비율·상각률 모두 정상권이므로 핵심 점검 대상은 아님.

2.2 데이터 출처 및 산출 룰

  • 원천: 금융감독원 DART 사업보고서 FY2025 (8개사 별도 재무제표 부속서류)
  • 기준일: 2025-12-31 별도 BS / FY2025 별도 PL
  • BS 핵심수치: 자산·자본·보험계약부채·이익잉여금 — 사업보고서 별도 재무제표 정합 확인
  • 변동표 13행: 사업보고서 「보험계약부채 변동분의 차이조정 공시」 표준 13행 구조 사용. 자사 미래에셋은 사업보고서 PDF disclosure 원본값, 7개사는 사업보고서 부속서류 표준 분해값을 표준 라벨로 cross-tab.
  • 위험률·사업비 마진: 각사 잔여기간 합계 위험보험료·예상보험금·예정유지비·예상유지비 (undiscounted, 잔여계약기간 합산)
  • 정합 검증: 기초 + Σ변동 = 기말 잔차 |%| < 2% 시 정합 OK 판정
  • 단위 통일: 본문 표시는 억원(억 = 108원). 1조 = 10,000억으로 환산 표기.

2.3 분석 방법론

  • 계리 흐름 순서: 부채(BEL/RA/CSM) → 손익(서비스결과·예실차·CSM 상각) → 마진(위험률·사업비) → 자본·감독적립 — 계리결산 실무 흐름과 동일하게 narrative 배열
  • 자사 위치 객관화: 모든 KPI 에 8사 최고/평균/최저 + 자사 값·순위 병기
  • 회사간 차이 해석: 사업 mix(보장성 vs 저축성), 회계정책 선택, 공시 깊이 차이를 우선 설명변수로 적용
  • 강점·점검 분리: 강점(green-box) · 점검(amber-box) · 한계(alert-box) 시각적 구분
본 보고서는 사업보고서·부속서류의 별도 재무제표 공시를 1차 source 로 사용하였으며, 일부 회사의 disclosure 깊이 차이로 인해 '미공시' 또는 '비교 불가' 표기되는 항목이 있음. PDF·구두값의 임의 입력은 적용하지 않았음.

8. 계리결산 시사점 및 권고

8.1 자사 강점 (유지·발신)

  1. 가정 정확도 (예실차 −0.0%, 8사 1위): 운영 단계 reconciliation 운영이 동업사 군집 중 최우수. 결산 적정성 검증 시 가장 신뢰 가능한 indicator. 계리포탈 분기별 자동 검증식으로 채택 가능.
  2. 사업비 보수성 (47.2%, 6공시사 1위): 예정유지비 대비 실제 추정치가 절반 수준 — 향후 사업비 차익 누적 잠재력 가장 큼. 사업비 운영 통제력의 정량 성과.
  3. 본업 마진 (서비스결과율 10.4%, 8사 1위): 단위 보험수익당 본업 마진이 동업사 최강. 사업비율 93.7% (8사 최저)에서 차익 6.3%p 확보 + RA 해소 비중 4.1% (8사 1위)의 마진 인식 효율이 결합된 결과.
  4. 변동 disclosure 완결성 (잔차 0): 보험계약부채 13행 변동표 기초+Σ변동=기말 완전 정합. 자사 disclosure 운영의 정확성이 동업사 군집 중 최우수.
  5. 가정변경 4종 분해 보고 깊이: 해지율·위험율·예정율·기타 4종 분해는 8사 중 자사만 완전 보고. 외부 stakeholder 와 감독당국에 대한 disclosure transparency 최강.

8.2 점검 시그널 (우선 조치 권고)

  1. 위험률 마진 103.5% (8사 7위): 생보 군집(78~90%) 대비 20%p 비보수. 특히 30년 초과 보장 가정 131.3% 가 합계 마진을 −1.2조 적자로 끌어내림. 권고: 종신·장기 보장성 상품의 위험률 재교정 또는 onerous 식별 분석 우선 수행.
  2. RA/BEL 1.50% (생보 4사 최저): 자사 RA 자본화 강도가 동업사 군집 대비 명확히 낮음. 권고: RA 산출 방식(분산 모형·신뢰수준)의 모형 가정 재검토. 동일 boundary 의 위험 자본화가 생보 군집 평균 2.06% 대비 약 0.5%p 부족.
  3. PL 보험금융손익 비중 8.66% (생보 4사 압도 최고): 자사가 PL 옵션을 적게 적용하거나 변동가능항목(VFA) 비중이 높아 금리·할인율 영향이 PL 에 집중 인식. 권고: 이익 변동성 측면에서 OCI option 적용 정책 점검.
  4. CSM 풀 정체 단계: 기시 → 기말 잔액 −1.0%, 신계약 CSM 인식이 당기상각 수준에 그침. 권고: 보장성 신계약 인식 활동 강화로 중장기 CSM 풀 확대 필요.

8.3 동업사 baseline 활용 권고

  • 한화생명 CSM 표준 분해: 8사 중 유일하게 CSM 변동표 표준 13행 완전 보고. 자사 CSM disclosure 보고 깊이 강화 시 한화생명 format 을 baseline 으로 채택 권고.
  • 동양생명 자산규모 매칭: 자본 1.53조·자산 군집 자사와 가장 유사. 정량 KPI 군집 외(외형 외) 본업 마진·예실차 비교 시 1:1 동업사 비교 가능.
  • 삼성생명 대형사 baseline: 위험률 마진 81.9% (생보 4사 최강) — 자사 위험률 재교정 시 보수성 reference 로 사용 가능.

8.4 결산 적정성 종합 판정

자사 FY2025 별도 계리결산은 운영 측 정확도와 사업비 보수성은 동업사 군집 중 최우수이나, 위험률 모형의 보수성과 RA 자본화 수준은 생보 군집 평균 대비 약점이 명확하다. 결산 적정성 자체에 즉시 문제 시그널은 없으나, 30년 초과 보장 가정과 RA 신뢰수준 재교정은 차분기 모형 점검 우선과제로 권고한다. CSM 절대규모 차이는 자산규모 종속변수로 결산 적정성 평가와 무관함.

  • CSM 변동표 표준 분해 한계: 8사 중 한화생명만 완전 보고. 자사 포함 7사는 일부 line(특히 신계약 CSM 인식·해지조정·가정변경) 표준 표시계층 미보고로, CSM 변동 line별 동업사 cross-tab 비교는 한화 baseline 대비 정성 비교만 가능.
  • 현대해상 별도 BS 잔액 보고 한계: 보험계약부채 단독 컨텍스트 보고가 부족 — 변동표 검증식 잔차 −6.4조 발생. 현대해상 BS 비율(CSM/부채·RA/부채)은 본 보고서에서 비교 제외.
  • RA 변동표 일부 line 미공시: 보험수익·취득현금흐름 행은 회사별 분해 단위 차이로 다수 '미공시'. RA 의 line별 cross-tab 깊이는 BEL 변동표 대비 제한적.
  • 가정변경 4종 분해: 자사 단독 보고. 동업사 4종 분해 매핑은 본 분기에서 불가 — 자사 가정변경 분해의 동업사 비교는 다음 분기 또는 동업사 PDF 별도 추출 필요.
  • 사업비 마진 미공시 2사: 삼성생명(예상유지비 미공시)·한화생명(예정·예상유지비 모두 미공시) — 6공시사 기준 비교만 제시.
  • 보증준비금 손보 4사 미공시: 변액보험 미운영으로 보증준비금 적립 의무 없음 — 생보 4사 단독 비교.

9.2 후속 작업 권고

  • P1 (차분기): 30년 초과 보장 가정의 위험률 재교정 분석 — 사망율·발생율 가정의 외부 통계(국민건강보험·생명표) 대비 검증.
  • P1 (차분기): RA 산출 모형(신뢰수준·분산 가정)의 재검토 — 생보 군집 평균 2.06% 수준의 RA/BEL 비율 달성 가능성 평가.
  • P2 (차분기): 한화생명 CSM 변동표 표준 분해 format 을 baseline 으로 자사 CSM disclosure 보고 깊이 강화 — 신계약 CSM 인식·assumption change 분해를 표준 표시계층으로 보고.
  • P2 (반기): PL 보험금융손익 8.66% (생보 4사 압도 최고) 의 원인 분석 — OCI option 적용 정책·변동가능항목 비중 별 분해.
  • P3 (연간): 동업사 가정변경 4종 분해 매핑 시도 — 동업사 사업보고서 PDF 별도 추출로 자사 4종 분해의 동업사 위치 객관화.
본 보고서 사용 시 caveat 요약
1) 모든 수치는 별도 BS · 원수 · 발행 보험계약 기준임. 연결·재보험 held 제외.
2) BEL 변동표 검증 잔차는 회사별 분해 단위 차이에 기인하며 공시 오류가 아님. 자사 잔차 0.
3) 예실차 −0.0% 는 보험서비스결과 = CSM 상각 + RA 해소 reconciliation 의 잔차 기준이며, 손실부담 손실(환입) minor 항목 포함시 미세 변동 가능.
4) 위험률·사업비 마진은 잔여계약기간 합계 undiscounted 기준이며, 회사별 잔여기간 분포 차이는 caveat. 자사 마진의 30년 초과 비중 38.5% (위험보험료 기준)임.

2자사 CSM 절대규모 최저, 부채 대비 비율은 정상권
자사 CSM 2.06조는 동업사 중 가장 낮음. 다만 CSM/보험계약부채 비율 7.6%는 생보 4사 군집(6.6~9.0%) 범위 내로 절대규모 차이는 회사 자산규모 차이의 종속변수임.
3자사 CSM 상각률 10.0% — 동업사 군집 내
자사 CSM 상각률 10.0%는 동업사 8.5~11.9% 군집 중간값권. §103 「당기서비스 변동 행 × CSM 컬럼」 셀(IFRS17 표준) 추출 결과로, 당기상각 = 2,058억.
4변동표 검증식 9개사 全社 잔차 발생 (DI817100 분해 깊이 차이)
기시+변동합=기말 검증식이 9사 모두 FAIL. 변동 line 중 일부만 XBRL fact로 보고되고 나머지는 axis 분해(상품군·계약군) context에 있어 main role 합산만으로 잔차 발생함. 분해 라인 합산 룰 보강 필요.
5흥국화재 DI817100 main role 0 element — 비교 부분 제외
흥국화재는 DI817100 (변동표) main role에 element 0개로 보고됨 (sub-section에만 분포). DI817105 잔액표는 보유함. 변동표 비교에서 흥국화재 제외, 잔액·라이프사이클 비교만 가능.

자사 관점 시사점

  • 자사 CSM 절대규모는 동업사 최저이나, 부채 대비 비율 기준으로는 군집 내 정상권으로 판단됨. 신계약 CSM 인식액 5,256억(자사) · 기시 CSM 대비 25.3% 갱신율로 자사 자산규모 대비 양호함.
  • 자사 가정변경 4종 분해(해지율·위험율·예정율·기타)는 9개사 중 가장 정합적으로 공시되어 있음. 다만 RA 영향은 자사도 4개사도 미공시로 동업사간 RA 산출방식 비교는 불가.
  • 자사 RA/BEL 비율 1.7%는 동업사 중 최저 수준(별도 검토자료)으로, 자사 보수성 수준 점검 시그널로 활용 가능함.

즉시 확인 이슈

P1 변동표 잔차 31,447억원 (자사) — DI817100 axis 분해 context(상품군별) 합산 누락 확인 필요.
P2 자사 보험계약자산(자산분) element 0개 — 손보 4사 및 삼성생명·한화생명은 보유. 자산분 인식 정책 차이 확인 필요.
P3 자사 RA/BEL 1.7% 보수성 적정성 — 익년 결산 시 RA 산출 시뮬레이션 결과 외부 동업사 분포 대비 위치 점검.

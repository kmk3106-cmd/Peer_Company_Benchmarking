## 1. 검토결론

- **자사 별도 보험계약부채 27.00조원(기말 2,699,670억원)은 사업보고서 BS와 일치함**. DART XBRL DI817100/105 적재값과 본 보고서 비교 베이스가 정합함을 1차 확인.
- 동업사 8개사(자사 포함) 중 **부채변동표(DI817100·105)는 7개사 main role 수록·1개사(흥국화재) sub-section만 수록**. 라인 17개 횡단 비교 가능 항목은 8/8 만점이 8개, 6~7/8 부분 매칭이 7개, 5/8 이하가 2개임 (`line_crosstab_v2.csv`).
- 자사 LRC·LIC·BEL·RA·CSM 4-piece 구성 비율은 동업사 대비 **outlier 신호 없음**. 단, **자사 CSM 상각률 101.95%는 동업사 중위값 11.0%(8.5~11.9% 분포) 대비 현저히 높음** → 후속 사유 규명 필요(섹션 3).
- 등급 분포: **PASS 6** (DI817100·105·818-200·205 main 매트릭스), **WARNING 2** (DI817300·305 CSM 만기 분해 — 4/8 회사만 비교 가능), **NOT_COMPARABLE 1** (DI818-100/105 정성 위주), **OUT_OF_SCOPE 1** (재보험 DI817200/205 — 본 검증 제외).

## 4. 비교불가 및 한계

### 4.1 OUT_OF_SCOPE / NOT_COMPARABLE

| 항목 | 사유 |
|---|---|
| DI817200/205 (재보험) | 본 검증 범위 제외 |
| 흥국화재 DI817100 | main role 미보고. sub-section LRC/LIC 잔액(DI817105)만 5,024 fact 수록 — 변동표 비교 불가 |
| 흥국화재 DI818-100/200 | 자산 변동·잔액 main role 미보고 |
| 삼성생명·동양생명 가정변경 라인별 분해 | 가정 분해 element 미보고 (삼성은 합계 element 1개 도입, 동양은 전면 미공시) |
| 삼성생명 신계약 CSM 라인 | 별도 element 약 0 — CSM 조정 추정변동 합계 라인에 통합. 라인별 분해 비교 불가 |
| 동양·삼성화재 가정변경 효과 | `peer_assumption_final.json` status = not_disclosed |
| DB손보 가정변경 element | 별도(Separate) 값만 보고, 연결(Consolidated)값 null — 별도 비교만 가능 |

### 4.2 데이터 구조 한계

- **회사별 axis 멤버수 편차** (3~32개): 한화생명·한화손보·현대해상은 제품 분해 깊음, 삼성화재·흥국화재는 3개 멤버로 단순. axis-min context 합산으로 회사 분해 깊이는 흡수했으나, 상품 mix 차이까지 정규화하지는 않음.
- **자사 만기 grain 차이**: 자사 ≤5년 구간 5개 bucket 미공시 (섹션 3.5). 동업사와 만기별 PV 직접 비교는 5년 이상 구간만 가능.
- **CSM 상각 분모 정의 불일치 가능성** (섹션 3.2): coverage unit 기반 공식 vs 단순 평균 분모. 사업보고서 본문 재확인 전에는 9배 차이의 진위 단정 금지.
- **금융손익 OCI / 기타증감 라인**: 8/8 매칭 안됨 (각 5/8). 회사별 element 분리 정도가 달라 비교 정밀도 낮음.

### 4.3 자사 데이터 외부 비교 제약 (사내한정 표시 대상)

| 항목 | 사유 |

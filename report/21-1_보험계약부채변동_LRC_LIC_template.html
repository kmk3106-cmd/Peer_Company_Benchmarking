표시 규칙 (Format Rules)

  1. 잔여보장부채(LRC)·발생사고부채(LIC) 분리 표기 — 각 상품군마다 LRC, LIC 2개 열로 구분.
  2. LRC = 손실요소외 + 손실요소 합산 — Loss Component를 별도 열로 분리하지 않고 LRC 한 열에 합산. 손실요소 인식·환입은 Ⅰ-2-4 행으로 표시되지만 열은 LRC에 귀속.
  3. 상품군별 분해 — 사망보장 · 건강보장 · 저축성 · 변액 · 기타 (생보 표준 5분류). 각 상품군은 색상으로 구분.
  4. 잔액 표시 순서 — 기초잔액은 항상 표 최상단, 기말잔액은 항상 표 최하단.

주석

  1. 본 표는 DART XBRL 역할 DI817100 (IFRS17 §104 LRC/LIC 변동분 차이조정)의 발행보험계약 부분만 포함합니다. 보유재보험은 DI817200에서 별도 공시합니다.
  2. 분류 축(DisaggregationOfInsuranceContractsAxis) = InsuranceContractsIssuedMember 만 포함, 재무제표 구분(ConsolidatedAndSeparateFinancialStatementsAxis) = SeparateMember(별도).
  3. 상품군 axis는 회사별 entity 확장 member 사용 — 표준 분류가 없어 라벨 매칭으로 후처리 필요.
  4. 금액 부호: 부채 증가(+), 부채 감소(−). 총합계 = LRC + LIC (대수합).
